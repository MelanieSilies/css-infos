rem verändert die Größe relativ zum Root Element (html)

  • 2rem = 32px
  • 0.5rem = 8px

Prozent

Eine Größenangabe in % gibt dem Element die Größe in Prozent des Elternelements.

  • 100% = 16 px
  • 200% = 32 px
  • 50% = 8px
