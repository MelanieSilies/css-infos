Position

Wenn wir die Art der Position mit Position definieren, können wir das Element frei platzieren.

Folgende Positionen können verändert werden, sowohl positiv, wie auch negativ:

  • top = Von oben nach unten
  • right = von rechts nach links
  • bottom = von unten nach oben
  • left = von links nach rechts

Bei position: relative; ändern wir die Position eines Elements relativ zu seiner Ausgangsposition.

bei position: absolute; platzieren wir das Element relativ zur absoluten position des Elternelements.

der Default-Wert von position: ; ist position:static; und sagt dem Element, dass es ein statischer Teil des html Baumes ist.

Randplatzierungsbeispiel

.positionbox kann genutzt werden, um Seiten "auszugrauen". Dies wird oft genutzt, wenn User etwas bestätigen müssen. Die Seite ist Sichtbar, aber nicht klickbar.

Position-Advanced

bei position:fixed; fixieren wir ein Element am Browserfenster. Es wird beim Scrollen nicht mehr bewegt.

bei position: sticky; fixieren wir den Inhalt an der Stelle im Browser, an dem wir sie Wollen. Sobald das Element vom browser scrollen eingeholt wird.

mit Z-Index: ; können wir die Positiom eines Elements auf der Tiefenachse (z) definieren, eine höhere Zahl, egal ob im positiven oder negativen Bereich. (Wichtig: Ein Element mit z-Index MUSS eine positionierungsangabe haben.)
